Polymorphism

Polymorphism is the ability of different classes to provide a unique implementation of the same method. In JavaScript, polymorphism can be achieved through method overriding.

Example:
Using the Animal and Dog classes from before, we demonstrate polymorphism by creating additional subclasses with their own implementations of the speak method.

class Animal { constructor(name) { this.name = name; } speak() { console.log(`${this.name} makes a sound.`); } } class Dog extends Animal { speak() { console.log(`${this.name} barks.`); } } class Cat extends Animal { speak() { console.log(`${this.name} meows.`); } } const animals = [ new Dog("Buddy"), new Cat("Whiskers") ]; animals.forEach(animal => animal.speak()); // Output: // "Buddy barks." // "Whiskers meows."

Both Dog and Cat are subclasses of Animal.
Each subclass provides its own implementation of the speak method, allowing each type of animal to "speak" in its way.
We use polymorphism by calling speak() on each object in the animals array, and JavaScript dynamically determines the correct implementation based on the object's type.
