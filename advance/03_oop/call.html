Call

The .call() function in JavaScript is a built-in method of functions that allows you to invoke a function with a specified this context and pass arguments individually.

functionName.call(thisArg, arg1, arg2, ...);

thisArg: Specifies the this value for the function when itâ€™s called. This is the object that this inside the function will refer to.
arg1, arg2, ...: The arguments you want to pass to the function, separated by commas.
When to Use .call()
The .call() method is useful when:
You need to invoke a function in the context of a specific object.
You want to borrow methods from other objects.
Example 1: Using .call() to Set this Context

const person = { firstName: "John", lastName: "Doe", fullName: function() { return this.firstName + " " + this.lastName; } }; const anotherPerson = { firstName: "Jane", lastName: "Smith" }; // Using call to invoke fullName with `anotherPerson` as `this` console.log(person.fullName.call(anotherPerson)); // "Jane Smith"

The fullName function is defined in person.
Using .call(), we invoke fullName with anotherPerson as the this context, so it returns "Jane Smith" instead of "John Doe".
